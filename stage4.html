Build Your First AI Project

In this project, we will build a **Handwritten Digit Recognition** AI using Python and TensorFlow.

Step 1: Install Dependencies

Ensure you have Python and TensorFlow installed. Run the following command:

pip install tensorflow numpy matplotlib

Step 2: Load the Dataset

Use the MNIST dataset, which contains handwritten digits (0-9).

import tensorflow as tf
from tensorflow import keras
import matplotlib.pyplot as plt

# Load dataset
mnist = keras.datasets.mnist
(x_train, y_train), (x_test, y_test) = mnist.load_data()

# Normalize data
x_train, x_test = x_train / 255.0, x_test / 255.0

Step 3: Build the AI Model

Create a simple neural network using TensorFlow.

model = keras.Sequential([
    keras.layers.Flatten(input_shape=(28, 28)),
    keras.layers.Dense(128, activation='relu'),
    keras.layers.Dense(10, activation='softmax')
])

model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

Step 4: Train the Model

Train the model with the dataset.

model.fit(x_train, y_train, epochs=5)

Step 5: Test the Model

Evaluate model performance on test data.

test_loss, test_acc = model.evaluate(x_test, y_test)
print("Test Accuracy:", test_acc)

Step 6: Make Predictions

Use the trained model to predict handwritten digits.

import numpy as np

predictions = model.predict(x_test)
plt.imshow(x_test[0], cmap=plt.cm.binary)
plt.title(f"Predicted: {np.argmax(predictions[0])}")
plt.show()
